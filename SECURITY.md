## Reporting a Vulnerability

If you discover a security vulnerability in SentinelX, please do **not** open a public GitHub Issue.

Instead, report it privately by emailing: jagdhariansh2004@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect a response within 72 hours.
